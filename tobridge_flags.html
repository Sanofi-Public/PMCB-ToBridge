Flags available for pre-processing (ToBridge)

FlagDefaultHelp
--cr_mkfastqFalseRun cellranger make fastq; make sure SampleSheet.csv is in your BCL folders
--bcl_convertFalseRun BCL convert; make sure SampleSheet.csv is in your BCL folders
--bcl_convert_sheet_convFalseConvert sample sheet to use BCL convert; use this flag if you have a 10x-style sample sheet
--fastqcFalseRun FASTQC
--star_soloFalseRun STARsolo
--star_solo_chemNoneWhat is the sample chemistry? If you run STARsolo, this must be provided. Example: SC3Pv3
--star_solo_featuresGeneFull_Ex50pASQuantification of different features; currently GeneFull_Ex50pAS (closest to CellRanger) and SJ (splicing analysis) have been tested. You can use them together as 'GeneFull_Ex50pAS SJ'
--star_solo_genome_generateFalseGenerate a reference genome for STARsolo
--star_solo_bamTrueGenerate sorted BAM files instead of SAMs
--cr_countFalseRun cellranger count
--cr_genome_generateNoneGenerate a new cellranger reference genome. Please provide a desired name
--cr_count_chemistryNoneSpecify chemistry for cellranger if cannot be detected automatically (rare)
--cr_count_forcecellsNoneForce a certain number of cells for cellranger count
--cr_count_bamFalseProduce BAM files upon alignment. Note that the default is to not produce BAMs
--cr_count_featureFalseFeature bacrode assay using cellranger
--alignment_qcFalseGenerates a QC report based on alignment results
--alignment_qc_genebodyFalseGenerates a QC report based on alignment results, which includes genebody coverage (slow)
--multi_qcTrueGenerates an HTML report based on all the QC
--sampAllIf you only want cellranger run on specific samples, use this flag, e.g. --samp S1,S5
--nthreadscpu_count-1Number of threads; defaults to all minus 1
